## Instructions to run

-   Pre-requisites:

    -   Node.js
    -   NPM

-   Directions to setup/install

```bash
$ git clone https://github.com/dsckiet/fashion-on-rent-backend
$ cd fashion-on-rent-backend
$ vim .env
# Set values in .env
$ npm i
```

-   Directions to execute

```bash
$ npm run dev
```
